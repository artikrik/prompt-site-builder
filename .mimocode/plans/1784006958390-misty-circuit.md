# Prompt Site Builder — ФІНАЛЬНИЙ ПЛАН

## Контракт

**Мета**: Зробити систему повністю робочою "під ключ" — від скрапінгу бізнесів, через збагачення даних та генерацію sales-скриптів, до створення та публікації сайтів з готовими шаблонами, AI-генерацією, можливістю редагування та інтеграціями з банкінгом.

**Доступні API ключі**: OpenAI + Google Maps

**Користувач**: Для особистого використання

**Рішення архітектури**:
- WayForPay/EasyWeek — вбудовані хостингові віджети (не API проксі)
- Шаблони + AI-генерація (обидва варіанти)
- Структурована форма для редагування сайтів
- API ключі зберігаються в БД через Settings UI

## ВЕРИФІКАЦІЯ

1. **Запуск**: `turbo dev` — бекенд та фронтенд стартують без помилок
2. **Scraping**: `POST /leads` → `POST /leads/:id/scrape` → перевірити scrapedData з реальними даними
3. **Enrichment**: `POST /leads/:id/enrich` → перевірити enrichmentData з competitors, reviews, salesScript
4. **Templates**: `GET /templates` → вибрати шаблон → `POST /templates/:name/generate` → перевірити publishedUrl
5. **Editor**: `GET /projects/:id/editor` → змінити секцію → `POST /projects/:id/regenerate` → перевірити зміни
6. **Addons**: Активувати WayForPay → відкрити сайт → перевірити що віджет рендериться
7. **Sales Script**: Відкрити Enrichment таб → Sales Script → перевірити всі 7 секцій
8. **CI**: `bash scripts/ci-local.sh` — lint, typecheck, test, build
